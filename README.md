
# ONE PIECE AVATAR

Consumo de la api https://api.jikan.moe/v4






## Run Locally

Clone the project

```bash
  git clone https://github.com/Delairi/onepiece_avatar.git
```


Install dependencies

```bash
  npm install
```

Start the frontend server

```bash
  npm run dev
```

Go to the backend directory

```bash
  cd backend
```

Install dependencies

```bash
  npm install
```

Start the backend server

```bash
  node --watch index.js
```
